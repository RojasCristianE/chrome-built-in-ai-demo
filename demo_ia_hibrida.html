Guía de Activación de Soberanía Cognitiva 💡

Para que las APIs nativas se ejecuten en tu navegador Chrome (sin requerir simulación), debes configurar los siguientes parámetros internos en tu equipo:

  1. Asegúrate de estar en **Google Chrome (versión 148+ para web o 138+ para extensiones)**.
  2. Accede a chrome://flags y activa **Prompt API for Gemini Nano** e **Enabling Optimization Guide On Device Model** (configúralo en Enabled BypassPrefGesture).
  3. Reinicia el navegador y visita chrome://components. Busca **Optimization Guide On Device Model** y pulsa "Buscar actualizaciones" para descargar físicamente a Gemini Nano (~1.5GB).
  4. Asegúrate de contar con al menos **10 GB de espacio libre** en disco para evitar que el navegador elimine el modelo local de forma automática.
